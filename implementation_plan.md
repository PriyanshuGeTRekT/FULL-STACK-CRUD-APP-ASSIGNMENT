# Implementation Plan

## Goal
Create a beginner-friendly full-stack CRUD app (React, Node, Mongo).

## Proposed Components
- **Backend**: Express server, Mongoose models, Nodemailer service.
- **Frontend**: Vite + React, Tailwind CSS, Axios, Chart.js.

## Verification
- Seed data verification.
- Manual testing of all CRUD operations.
- Email delivery check (Ethereal).
